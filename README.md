# Hollow County

Hollow County is an original top-down procedural zombie survival game that runs directly in a web browser. Every seed creates different roads, buildings, searchable interiors, loot, forests, ponds, wrecked vehicles, and zombie placement.

## The four required files

Keep all four files together at the top level of your GitHub repository:

```text
index.html
styles.css
game.js
README.md
```

Do not place them inside another folder. The complete game code is bundled inside `game.js`.

## Publish with GitHub Pages

1. Create a public GitHub repository named `hollow-county`.
2. Create each of the four files above and paste in its matching contents.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.
6. The game will appear at `https://tyronhuggins32-cmd.github.io/hollow-county/`.

## Controls

- `WASD`: Move
- `Shift`: Sprint
- `C`: Sneak
- `E`: Search
- Left click: Aim and attack
- `Tab`: Inventory
- `B`: Craft and build
- `1`–`5`: Select hotbar item
- `Escape`: Pause
- Phone: Use the on-screen joystick and action buttons

## Included gameplay

- Seeded procedural county generation
- Enterable houses, stores, clinics, police buildings, and warehouses
- Searchable containers and randomized loot
- Health, stamina, hunger, thirst, weight, and infection
- Zombie sight, hearing, investigation, pursuit, memory, and pathfinding
- Melee weapons, guns, ammunition, noise, and hordes
- Inventory, hotbar, crafting, barricades, and noise traps
- Day/night lighting, weather, flashlight, autosaving, and continue
- Desktop and phone controls

All code and visuals are original and use no external game assets.
